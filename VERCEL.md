# Deploy on Vercel

This package is configured as a standard Next.js application.

1. Extract the ZIP and upload the complete folder contents to a GitHub repository.
2. Import that repository in Vercel.
3. Keep **Framework Preset** set to **Next.js** and **Root Directory** set to `.`.
4. Keep the default install command. The build command is `pnpm run build`.
5. No environment variables are required for the current public educational site.

The project pins Node.js 22 and pnpm 10.28.0 for repeatable builds.
